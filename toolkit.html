NotebookLM

NotebookLM is an AI-powered research assistant designed for academic work. It grounds its responses exclusively in the sources you upload (such as PDFs, Docs, and websites), providing inline citations to minimise hallucination and support rigorous scholarship.

Google AI Studio

Google AI Studio is a web-based platform for experimenting with and building applications on top of Google's Gemini family of models. It provides an intuitive interface for academics to prototype ideas, generate code, and analyse multimodal data.

Core Functionalities

    [cite_start]
  • Advanced Text Analysis: Summarise papers, draft grant proposals, and generate educational content using various prompt styles. [cite: 33]
    • [cite_start]
  • Multimodal Capabilities: Process and analyse information from images, audio, and video for innovative research. [cite: 33]
    • [cite_start]
  • Code Generation: Describe a function in natural language to rapidly generate code in various programming languages, ideal for prototyping tools and automating pipelines. [cite: 33]
